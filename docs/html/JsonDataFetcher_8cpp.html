Detailed Description

Periodically fetches JSON data from a URL and writes it to a file.

This program uses libcurl to fetch data and nlohmann::json to parse it. It is intended to collect real-time vehicle data (e.g., bus positions).

Compile with: g++ -std=c++14 -o JsonDataFetcher JsonDataFetcher.cpp -lcurl
