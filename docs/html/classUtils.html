Calculates the distance from a bus stop to a trip shape's coordinates.

This method calculates the distance between a given bus stop's coordinates and the closest point on the bus trip's shape.

Parameters
tripShapeA map of trip shapes, where each shape is associated with a list of coordinates.
shape_idThe identifier of the bus trip shape.
stop_coordinatesThe coordinates of the bus stop.
Returns
The distance to the bus stop from the trip's shape, in meters.
Exceptions
std::runtime_errorif the shape ID is not found or the coordinates are invalid.
